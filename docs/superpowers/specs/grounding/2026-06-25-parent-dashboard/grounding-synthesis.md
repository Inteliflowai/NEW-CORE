- Generic primitives (`src/lib/copy/leakGuard.ts`, pure): `hasLeak`/`assertNoLeak` (bans ALL digits `/\d/`, %, avg, score N, percentile, rank); `hasBannedWord`/`assertNoBannedWord` (score/percentile/index/divergence/threshold/signal/model/algorithm/flag). **`'risk'` is INTENTIONALLY excluded (allowed for teachers).**
- **REUSE `src/lib/highfives/guardrail.ts` `FOUR_AUDIENCE_LEAKS`** — already catches band/mastery vocab (advanced/proficient/reteach/mastery/band/basic/remedial/"grade level"), peer comparisons (top of the class/ahead of/classmates/than half/rest/others), letter grades, spelled-out "percent".
