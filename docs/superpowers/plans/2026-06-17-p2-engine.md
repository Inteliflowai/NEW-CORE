# CORE v2 P2 — Generation Engine Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (- [ ]) syntax.

**Goal:** Lift V1's five-call generation pipeline (lesson parse → quiz gen → within-attempt adapt → OEQ grading → differentiated assignment) onto the existing Foundation spine as **import-safe `src/lib/engine/*` functions** that the route handlers AND the eval rig both import, with prompt **text** ported verbatim and the grading path defended by a week-1 grader spike.

**Architecture:** Each AI path becomes a pure, headless `src/lib/engine/<call>.ts` function (no `next/server`, no `cookies()`, no SDK side-effects at import) that takes plain data in and returns Zod-validated data out; the API route handler does auth + DB + metering around it, and `scripts/eval/runner.ts`'s `invokeCandidate` calls the same function directly. Default execution is **direct awaited calls with retry/idempotency** (the proven path); Vercel Workflow DevKit is a week-1 spike with the awaited path as the committed default. Prompt text lifts verbatim from V1 `lib/openai/prompts.ts`; the LLM wrappers and model registry come from the Foundation plan's `src/lib/ai/*`.

**Tech Stack:** Next.js 16.2.9 · React 19.2.4 · TypeScript 5 · `openai` (GPT-4o generation) · `@anthropic-ai/sdk` (Claude grading) · `zod` (output validation) · `unpdf` + `mammoth` (upload parsing) · Vitest · the Foundation `src/lib/{ai,supabase,auth}` modules + `scripts/eval` harness.

## Global Constraints (project-wide rules — copy verbatim; obey)

- **Stack is locked:** Next 16.2.9 + React 19.2.4 + TS5. `@/*` resolves to `./src/*`.
- **Turbopack API trap — nest under existing stub paths only.** The Foundation plan created the `src/app/api/**` route-stub tree. Generation routes here MUST be **nested under existing stub paths** (`api/teacher/lessons/parse`, `api/attempts/[attemptId]/{adapt,submit}`, `api/teacher/assignments/generate`, `api/teacher/quizzes/generate`); fill stub bodies — do NOT create new top-level `api/` folders mid-build. Restart the dev server after adding any new top-level segment.
- **Auth:** `auth.getUser()` for trust decisions, never `getSession()`. `cookies()` is async in Next 16 — always `await cookies()`. Route handlers use `createServerSupabaseClient()` (session/RLS) for the auth check and `createAdminSupabaseClient()` (service-role) for engine writes; **every cross-user admin-client read binds an object-level guard** from `src/lib/auth/guards.ts` (`guardClassAccess`, `guardStudentAccess`) — RLS is NOT the backstop on admin-client paths.
- **Model IDs come from `src/lib/ai/models.ts` only — never hardcode at a call site.** `CLAUDE_GRADING_MODEL` (default `claude-sonnet-4-6`) for grading; `OPENAI_GEN_MODEL` (default `gpt-4o`) for lesson parse / quiz / adapt / assignment / learning-style. **Opus is a week-1 SPIKE only** — the registry default stays Sonnet/GPT unless the spike (Task 7) passes; flipping it is a one-line env change.
- **`lib/engine` fns must be import-safe:** no `next/server`, no `cookies()`, no module-load-time SDK construction that throws — so routes AND the `tsx`-run eval rig share them headless. Construct/inject clients via the Foundation wrappers (`resilientChatCompletion`, `claudeChat`) which already guard missing keys.
- **Never the word "Band" in UI.** The DB enum stays `reteach|grade_level|advanced`; the teacher-facing verbs are Reinforce / On Track / Enrich. Engine functions emit the raw `mastery_band` enum value internally; UI mapping is downstream. Differentiation here uses the **quiz-score mastery band** (`computeMasteryBand`), NOT per-skill CL (Plan 3).
- **Grading never half-grades and never fabricates a score.** If any concurrent OEQ grading call returns null/throws, the whole submit short-circuits to `grading_status:'pending'` + `grading_failed:true` (V1's proven invariant, §3.5 terminal-failure contract). - **Grade-anchored difficulty (Bloom-to-grade) is a hard constraint.** The quiz prompt's Bloom-to-grade calibration block (`prompts.ts:367–372`) ports verbatim; the grade comes from the lesson data, never a difficulty slider.
- **SCOPE.md wins** over any divergence here or in the design spec — flag divergences as an open risk, never silently diverge.

// src/lib/utils/scoring.ts
// LIFT V1 lib/utils/scoring.ts (band thresholds — single source, never re-hardcode).
export type MasteryBand = 'reteach' | 'grade_level' | 'advanced';

/** ≤50 reteach / 51–79 grade_level / ≥80 advanced (V1 scoring.ts:9–12). */
export function computeMasteryBand(scorePct: number): MasteryBand {
  if (scorePct <= 50) return 'reteach';
  if (scorePct <= 79) return 'grade_level';
  return 'advanced';
}

/** Combine 3 MCQ (0/1 each) + 2 OEQ (0/0.5/1) into a 0–100 percent.
 *  LIFT V1 computeFinalScore — MCQ weight = OEQ weight per V1's locked scaling. */
export function computeFinalScore(
  mcqScores: number[],
  openScores: number[],
): { rawScore: number; scorePct: number } {
  const rawScore = mcqScores.reduce((a, b) => a + b, 0) + openScores.reduce((a, b) => a + b, 0);
  const maxScore = mcqScores.length + openScores.length; // 3 + 2 = 5
  const scorePct = maxScore > 0 ? Math.round((rawScore / maxScore) * 100) : 0;
  return { rawScore, scorePct };
}

/** Serverless-safe PDF extraction via unpdf (no canvas/workers) — LIFT V1 lib/parsePdf.ts. */
async function extractPdfText(buffer: Buffer): Promise<string> {
  const { extractText } = await import('unpdf');
  const { text } = await extractText(new Uint8Array(buffer));
  return text.join('\n\n');
}

export async function extractUploadText(
  buffer: Buffer,
  fileType: string,
  fileName: string,
): Promise<string> {
  const ext = fileName.split('.').pop()?.toLowerCase() || '';
  if (fileType === 'application/pdf' || ext === 'pdf') {
    return extractPdfText(buffer);
  }
  if (
    fileType === 'application/vnd.openxmlformats-officedocument.wordprocessingml.document' ||
    ext === 'docx'
  ) {
    const mammoth = (await import('mammoth')).default;
    const result = await mammoth.extractRawText({ buffer });
    return result.value || '';
  }
  // txt, md, or other text formats
  return buffer.toString('utf-8');
}

${LEVEL_INSTRUCTIONS[level]}

Lesson topic: ${input.lessonContext}

Original Q4: ${input.originalQ4 || 'N/A'}
Original Q5: ${input.originalQ5 || 'N/A'}

Generate 2 personalized open-response questions for this student. Return JSON:

    ],
    temperature: 0.7,
    max_tokens: 1200,
    response_format: { type: 'json_object' },

// src/lib/engine/grading.ts
// Engine call #4 — HIGHEST stakes (import-safe). LIFT V1 submit/route.ts:245–249 grading
// call. Claude primary (claudeChat, temp 0.2, 600 tok) → GPT fallback. On both exhausted,
// throws LlmExhaustedError — the attempt stays ungraded + re-queueable (§3.5). A blank/idk
// response is a real 0.0 grade FROM the model, never a synthesized default.
//
// CALIBRATION NOTE: CLAUDE_GRADING_MODEL defaults to claude-sonnet-4-6 (V1-proven, corpus-
// calibrated). If the week-1 grader spike (Task 7) selects an Opus 4.x model, the Claude
// request SHAPE must be rebuilt (drop temperature, use output_config.format structured
// output, handle stop_reason:'refusal' before reading content) — the prompt TEXT is unchanged.
// The wrappers in @/lib/ai are the seam where that rebuild lands; this fn calls them unchanged.
import { claudeChat } from '@/lib/ai/claude';
import { resilientChatCompletion } from '@/lib/ai/openai';
import { OPENAI_GRADING_FALLBACK } from '@/lib/ai/models';
import { GRADING_SYSTEM, gradingPrompt } from '@/lib/openai/prompts';
import { GradingResultSchema, type GradingResult } from '@/lib/engine/types';
import { LlmExhaustedError } from '@/lib/ai/errors';

export interface GradeInput {
  questionText: string;
  rubric: string;
  response: string;
  rubricVersion?: string;
}

const SYSTEM = GRADING_SYSTEM + '\nReturn ONLY valid JSON. No markdown code fences.';

export async function gradeOpenResponse(input: GradeInput): Promise<GradingResult> {
  const userPrompt = gradingPrompt(input.questionText, input.rubric, input.response || '', input.rubricVersion ?? 'v1');

  // ── Primary: Claude (calibration-locked, temp 0.2) ──
  const claudeRaw = await claudeChat(SYSTEM, userPrompt, { temperature: 0.2, maxTokens: 600 });
  if (claudeRaw) {
    const parsed = tryParse(claudeRaw);
    if (parsed) return parsed;
  }

  // ── Fallback: GPT ──
  const gptCompletion = await resilientChatCompletion({
    model: OPENAI_GRADING_FALLBACK,
    messages: [{ role: 'system', content: SYSTEM }, { role: 'user', content: userPrompt }],
    temperature: 0.2,
    max_tokens: 600,
    response_format: { type: 'json_object' },
  });
  const gptRaw = gptCompletion?.choices[0]?.message?.content;
  if (gptRaw) {
    const parsed = tryParse(gptRaw);
    if (parsed) return parsed;
  }

  // Both exhausted (or both produced unparseable output) — terminal, never fabricate.
  throw new LlmExhaustedError('claude+openai');
}

function tryParse(raw: string): GradingResult | null {
  try {
    const result = GradingResultSchema.safeParse(JSON.parse(raw));
    return result.success ? result.data : null;
  } catch {
    return null;
  }
}

        return { task: t, grade: null };
      }

    // NEVER half-grade — if ANY OEQ failed, mark pending and bail (V1 submit:261–273).
    if (results.some(r => !r.grade)) {

        submitted_at: new Date().toISOString(), is_complete: true,
        grading_failed: true, grading_status: 'pending',

        message: 'Your answers have been saved. Grading is temporarily delayed — check back shortly.',

    for (const { task, grade } of results) {

    }).eq('id', attemptId);

import { claudeChat } from '@/lib/ai/claude';
import { resilientChatCompletion } from '@/lib/ai/openai';
import { OPENAI_GEN_MODEL } from '@/lib/ai/models';
import {

export interface AssignmentInput {
  lessonSummary: string;
  band: 'reteach' | 'grade_level' | 'advanced';  // the quiz-score mastery band (Plan-3 CL is separate)
  style: string;
  studentName: string;
  sparkEnabled?: boolean;
  targetedPractice?: boolean;

export async function generateAssignment(input: AssignmentInput): Promise<Assignment> {
  const strategies = getStrategiesForStudent(input.band, input.style).map(s => ({
    name: s.name, what_students_do: s.what_students_do, atl_skills: s.atl_skills,
    ib_learner_profile: s.ib_learner_profile, bloom_level: s.bloom_level,
  }));
  const userPrompt = assignmentPrompt(
    input.lessonSummary, input.band, input.style, input.studentName,
    strategies, input.sparkEnabled, input.targetedPractice,
  );

  // ── Primary: Claude (temp 0.7, large max_tokens, long timeout for streaming) ──
  const claudeRaw = await claudeChat(ASSIGNMENT_SYSTEM, userPrompt, { temperature: 0.7, maxTokens: 4500, timeoutMs: 120000 });
  if (claudeRaw) {
    const parsed = tryParseAssignment(claudeRaw);
    if (parsed) return parsed;
  }

  // ── Fallback: GPT ──
  const gpt = await resilientChatCompletion({
    model: OPENAI_GEN_MODEL,
    messages: [{ role: 'system', content: ASSIGNMENT_SYSTEM }, { role: 'user', content: userPrompt }],
    temperature: 0.7,
    max_tokens: 4500,
    response_format: { type: 'json_object' },
  }, { timeoutMs: 45000 });
  const gptRaw = gpt?.choices[0]?.message?.content;
  if (gptRaw) {
    const parsed = tryParseAssignment(gptRaw);
    if (parsed) return parsed;
  }

  throw new LlmExhaustedError('claude+openai');
}

function tryParseAssignment(raw: string): Assignment | null {
  try {
    const result = AssignmentSchema.safeParse(JSON.parse(raw));
    return result.success ? result.data : null;
  } catch {
    return null;
  }
}

/** #5a — infer learning style from behavioral signals. GPT-only; null → emerging (V1 submit:477). */
export async function inferLearningStyle(signals: string): Promise<LearningStyle> {

    model: OPENAI_GEN_MODEL,
    messages: [{ role: 'system', content: LEARNING_STYLE_SYSTEM }, { role: 'user', content: learningStylePrompt(signals) }],
    temperature: 0.3,
    max_tokens: 300,
    response_format: { type: 'json_object' },
  });

    const result = LearningStyleSchema.safeParse(JSON.parse(raw));

  } catch {
    return { learning_style: 'emerging', confidence: 0 };
  }
}

    const lesson = (attempt.quizzes as { lessons: { parsed_content: unknown; title: string } }).lessons;

    const assignment = await generateAssignment({

      band, style: learning_style || 'emerging', studentName,
    });

    const { data: row, error: insErr } = await admin.from('assignments').insert({
      quiz_attempt_id: attempt.id, student_id: attempt.student_id, class_id: attempt.class_id,
      lesson_id: (attempt.quizzes as { lesson_id: string }).lesson_id,
      mastery_band: band, learning_style: learning_style || 'emerging',
      content: assignment, status: 'draft', generation_model: OPENAI_GEN_MODEL,
    }).select().single();
    if (insErr || !row) return NextResponse.json({ error: 'Failed to save assignment' }, { status: 500 });

    return NextResponse.json({ assignment_id: row.id, content: assignment });
  } catch (err) {
    console.error('[teacher/assignments/generate] error:', err);
    return respondEngineError(err);
  }
}

// DEFAULT durable execution = direct awaited calls with per-step retry (inside each
// engine fn's wrapper) + idempotency (skip a step whose output is already cached).
// WDK is a week-1 SPIKE only (docs/spikes/wdk-spike.md); the awaited path is what ships

import { parseLesson } from '@/lib/engine/lessonParse';

export interface PipelineInput {
  lessonText: string;
  parsedLesson?: ParsedLesson;   // idempotency: if present, the parse step is already done
  quiz?: GeneratedQuiz;          // idempotency: if present, the quiz step is already done
}
export interface PipelineResult {
  parsedLesson: ParsedLesson;
  quiz: GeneratedQuiz;
}

export async function runGenerationPipeline(input: PipelineInput): Promise<PipelineResult> {

  // Step 2 — quiz gen (skip if cached).

  return { parsedLesson, quiz };
}
